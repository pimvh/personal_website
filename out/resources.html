Dataveiligheid - Have I been PWND?

Have I been PWND? is een open database op het internet waarin je je eigen email kunt opzoeken. Deze database op basis van vrijgegeven gegevens door hackers.

De tool is handig om op te zoeken welke recente hacks er zijn geweest en of je actie moet ondernemen in het veranderen van je wachtwoorden. Tegenwoordig zijn bedrijven verplicht om aan te geven dat gebruikers gehackt zijn, maar dat gebeurde vroeger niet altijd. Deze tool helpt daarbij.

Firefox Send - een gratis veilige bestand verstuurder.

Middels Firefox Send kun je bestanden ter grootte van maximaal 1GB (2.5GB als je inlogt op je firefox account) veilig versturen naar anderen via een link.

De meeste bestand verstuurders bieden opties als, automatisch verwijderen na x dagen of x downloads niet standaard aan, dat maakt deze bestandverstuurder erg handig.

Mozilla Privacy Not Included - Een gids voor privacyveilige aankopen.

De Mozilla Privacy buyer guide laat een ander perspectief zien op een aantal aankopen. In deze gids kun je met verschillende filters verschillende soorten koptelefoons, camera's en andere apparaten laten zien.

De website hanteert een creepy-schaal voor hoe angstaanjagend andere gebruikers een product vinden.

Online vergadertool - Jitsi.

Jitsi is een open source tool waarmee je gratis online kunt vergaderen en videobellen. Zonder nieuwe software, zoals Skype of andere videobel-tools, te installeren kan men eenvoudig met elkaar praten.

De Tool werkt prima op meerdere browsers en videocalls zijn eenvoudig te beveiligen met een wachtwoord.

Privacytools.io - Bron voor online privacy.

De website privacytools.io is een goede start om je internetprivacy naar een hoger niveau te brengen. Verder worden op deze website verschillende argumenten geven waarom internetprivacy van belang is (e.g. een antwoord op 'Maar ik heb niets te verbergen').

Je vindt op deze website een gedetailleerde lijst van services/tips over verschillende services (browsers, emailclients, messaging apps, etc.) die beter rekening houden met je privacy.

Bekijk, als je gebruiker van een firefoxbrowser bent, zeker het onderdeel waar verschillende add-ons voor firefox worden aangeraden. Deze zijn allemaal vrij eenvoudig aan je browser toe te voegen.

Programmeertaal - Haskell.

Op Learn you a haskell is een gratis boek (ook te kopen) te vinden waarmee je Haskell online kunt leren. Haskell is een lui-evaluerende pure functionele programmeertaal.

Het is bijvoorbeeld mogelijk om een eindeloze lange lijst (i.e. van 1 tot oneindig) te definiëren, maar ook meervoudige list-comprehension te schrijven om rechthoekige driehoeken te zoeken met een omtrek van 24 cm: let triangles = [(a, b, c) | a <- [1..20], b <- [1..20], c <- [1..20], a + b + c == 24, a^2 + b^2 == c^2]

Al met al een interessante programmeertaal.

Rank a Brand - Merken vergelijken qua duurzaamheid.

De website Rank a Brand is een website, grotendeels op basis van vrijwilligers, die de duurzaamheid van allerlei soorten merken test.

De labels kloppen niet altijd (of zijn niet altijd beschikbaar), maar mocht je van plan zijn om nieuwe spullen te kopen, dan is dit altijd een goede website om van tevoren te even te controleren.

Security Checklist - Wees veilig op het internet.

Met Security Checklist loop je een simpele afvinklijst door om je internetervaring beter te beveiligen, evenals privacytools.io raden zij verschillende passwordmangers, 2-factor authentication en verschillende apps aan. Daarnaast staan er bronnen onder elke stap, met verdere uileg over bijvoorbeeld het instellen van toegangsrechten voor verschillende apps.

Todoist - Handige takenlijst.

De Takenlijst app Todoist, is handig om je taken bij te houden. De kosten van deze app blijven erg beperkt (~15 eu per jaar). De app is ook multiplatform (helaas niet op Ubuntu) en via het internet beschikbaar.

Hiernaast heeft de website verschillende tutorials/templates voor het maken van lijstjes om je gestelde doelen te behalen.
